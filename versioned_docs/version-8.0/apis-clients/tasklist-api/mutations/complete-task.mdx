Complete a task with taskId and optional variables. Returns the task.

```graphql
completeTask(
  taskId: String!
  variables: [VariableInput!]!
): Task!

```

### Arguments

#### `taskId` ([`String`](/docs/apis-clients/tasklist-api/scalars/string))

#### `variables` ([`VariableInput`](/docs/apis-clients/tasklist-api/inputs/variable-input))

### Type

#### [`Task`](/docs/apis-clients/tasklist-api/objects/task)

Describes the User task.
