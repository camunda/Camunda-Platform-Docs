### Literal

Creates a new numeric value. Leading zeros are valid.

```feel
1

0.5
.5

-2

01

-0002
```

### Addition

```feel
2 + 3
// 5
```

### Subtraction

```feel
5 - 3
// 2
```

### Multiplication

```feel
5 * 3
// 15
```

### Division

```feel
6 / 2
// 3
```

### Exponentiation

```feel
2 ** 3
// 8
```
