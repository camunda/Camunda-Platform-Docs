Get a collection of Variables by name

```graphql
variables(
  taskId: String!
  variableNames: [String!]!
): [Variable!]!

```

### Arguments

#### `taskId` ([`String`](/docs/apis-tools/tasklist-api/scalars/string))

#### `variableNames` ([`String`](/docs/apis-tools/tasklist-api/scalars/string))

### Type

#### [`Variable`](/docs/apis-tools/tasklist-api/objects/variable)

Variable used in task.
