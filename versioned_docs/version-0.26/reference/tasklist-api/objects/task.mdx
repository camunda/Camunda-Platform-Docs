Describes the User task

```graphql
type Task {
  id: ID!
  name: String!
  workflowName: String!
  creationTime: String!
  completionTime: String
  assignee: User
  variables: [Variable!]
  taskState: TaskState!
}
```

### Fields

#### `id` ([`ID!`](/reference/tasklist-api/scalars/id.mdx))

#### `workflowName` ([`String!`](/reference/tasklist-api/scalars/string.mdx))

#### `creationTime` ([`String!`](/reference/tasklist-api/scalars/string.mdx))

When was the task created

#### `completionTime` ([`String`](/reference/tasklist-api/scalars/string.mdx))

#### `assignee` ([`User`](/reference/tasklist-api/objects/user.mdx))

Who is assigned to the task

#### `variables` ([`[Variable!]`](/reference/tasklist-api/objects/variable.mdx))

Variables associated to the task

#### `taskState` ([`TaskState!`](/reference/tasklist-api/enums/task-state.mdx))

State of the task
