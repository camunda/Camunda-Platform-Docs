Describes the User task.

```graphql
type Task {
  id: ID!
  name: String!
  processName: String!
  creationTime: String!
  completionTime: String
  assignee: User
  variables: [Variable!]
  taskState: TaskState!
  sortValues: [String!]
  isFirst: Boolean
  formKey: String
  processDefinitionId: String
}
```

### Fields

#### `id` ([`ID!`](/reference/tasklist-api/scalars/id.mdx))

The unique identifier of the task

#### `name` ([`String!`](/reference/tasklist-api/scalars/string.mdx))

Name of the task

#### `processName` ([`String!`](/reference/tasklist-api/scalars/string.mdx))

Name of the process

#### `creationTime` ([`String!`](/reference/tasklist-api/scalars/string.mdx))

When was the task created

#### `completionTime` ([`String`](/reference/tasklist-api/scalars/string.mdx))

When was the task completed

#### `assignee` ([`User`](/reference/tasklist-api/objects/user.mdx))

Who is assigned to the task

#### `variables` ([`[Variable!]`](/reference/tasklist-api/objects/variable.mdx))

Variables associated to the task

#### `taskState` ([`TaskState!`](/reference/tasklist-api/enums/task-state.mdx))

State of the task

#### `sortValues` ([`[String!]`](/reference/tasklist-api/scalars/string.mdx))

Array of values to be copied into `TaskQuery` to request for next or previous page of tasks.

#### `isFirst` ([`Boolean`](/reference/tasklist-api/scalars/boolean.mdx))

Flag to show that the task is first in current filter

#### `formKey` ([`String`](/reference/tasklist-api/scalars/string.mdx))

Reference to the task form

#### `processDefinitionId` ([`String`](/reference/tasklist-api/scalars/string.mdx))

Reference to process definition
