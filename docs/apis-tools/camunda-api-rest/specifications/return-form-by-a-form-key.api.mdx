<h1 className={"openapi__heading"}>Return form by a Form Key.</h1>

<MethodEndpoint method={"get"} path={"/forms"}></MethodEndpoint>

Get the form by the form key.

:::note
This endpoint will return only linked forms, Embedded forms are not retrieved for this endpoint.
:::

:::note
This endpoint is an alpha feature and not enabled on Camunda clusters out of the box.
See the [Camunda 8 REST API overview](/apis-tools/camunda-api-rest/camunda-api-rest-overview.md#query-api)
for further details.
:::
