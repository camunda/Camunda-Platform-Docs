<h1 className={"openapi__heading"}>Get process start form (alpha)</h1>

<MethodEndpoint
  method={"get"}
  path={"/process-definitions/{processDefinitionsKey}/form"}
></MethodEndpoint>

Get the start form of a process.

Note that this endpoint will only return linked forms. This endpoint does not support embedded forms.
Furthermore, this endpoint is an [alpha feature](/components/early-access/alpha/alpha-features.md) and not enabled on Camunda clusters out of the box.
The [Camunda 8 API (REST) Overview page](/apis-tools/camunda-api-rest/camunda-api-rest-overview.md#query-api) provides further details.

:::note
This endpoint is an [alpha feature](/components/early-access/alpha/alpha-features.md) and may be subject to change
in future releases.
:::
