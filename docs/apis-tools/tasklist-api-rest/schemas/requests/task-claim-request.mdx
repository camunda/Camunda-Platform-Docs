### Fields

#### [<code style={{ fontWeight: 'normal' }}>TaskClaimRequest.<b>assignee</b></code>](#)<Bullet />`string` <Badge class="secondary" text="scalar"/>

> When using a `JWT` token, the assignee parameter is NOT optional when called directly from the API. The system will not be able to detect the assignee from the JWT token, therefore the assignee parameter needs to be explicitly passed in this instance.

#### [<code style={{ fontWeight: 'normal' }}>TaskClaimRequest.<b>allowOverrideAssignment</b></code>](#)<Bullet />`boolean` <Badge class="secondary" text="scalar"/>

> When `true` the task that is already assigned may be claimed again. Otherwise, the task must be first unclaimed and only then claimed again. (Default: `true`).
