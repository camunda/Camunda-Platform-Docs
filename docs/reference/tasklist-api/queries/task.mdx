Get one task by id. Returns task or error when task does not exist.

```graphql
task(
  id: String!
): Task!

```

### Arguments

#### `id` ([`String!`](/docs/reference/tasklist-api/scalars/string))

### Type

#### [`Task`](/docs/reference/tasklist-api/objects/task)

Describes the User task.
