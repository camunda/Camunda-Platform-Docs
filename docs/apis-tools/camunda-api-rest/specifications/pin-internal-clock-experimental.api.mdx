<h1 className={"openapi__heading"}>Pin internal clock (experimental)</h1>

<MethodEndpoint method={"put"} path={"/clock"}></MethodEndpoint>

Set a precise, static time for the Zeebe engine’s internal clock.
When the clock is pinned, it remains at the specified time and does not advance.
To change the time, the clock must be pinned again with a new timestamp.

:::note
This endpoint is experimental. It may undergo changes or improvements in future releases.
:::
